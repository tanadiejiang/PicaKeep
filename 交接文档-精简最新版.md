# PicaKeep 交接文档 - 精简最新版

> 功能地图，2 分钟了解项目全貌。需要修改代码时，务必先查对应模块的"详见经验版"锚点。

## 项目定位

PicaKeep 是一个以本地漫画/本地图集/已下载管理为主线，同时支持客户端/服务端双模式的 Flutter 项目。当前版本已稳定运行，`flutter analyze` 0 issues，`flutter build apk --debug` 通过。

---

## 模块功能清单

### 远程服务与连接池

- 远程图片并发加载：阅读闸（默认 6）/ 浏览闸（默认 8），可在设置-阅读/浏览中 1~12 实时调节
- 连接池分流：图片 bulk 池（24 连接）与 JSON 控制池（6 连接）隔离，封面走 bulk 池
- 连接池泄漏保护：孤儿连接超时回收 + 非 2xx 响应体 drain + body idle timeout 守卫 ⚠️ **修改图片/控制面请求链路必须读 [经验版 §1](./交接文档-经验思路版.md#1-远程服务与连接池)**
- 封面磁盘缓存：带失败负缓存（404 冷却 10 分钟），自动按大小限制裁剪旧文件

### 收藏 / 图片收藏远程档

- 收藏页远程档：列出远程收藏文件夹，支持中文路径，点击进阅读
- 图片收藏远程档：网格展示缩略图，无图项静默破图占位，点击定位资源进阅读
- 远程可用性判断、档位开关无条件显示 ⚠️ **修改收藏/图片收藏远程链路必须读 [经验版 §2](./交接文档-经验思路版.md#2-收藏图片收藏远程档)**

### 加密压缩包

- ZIP/CBZ 内部章节名编码修复：支持 GBK/Shift-JIS/UTF-8 自动识别
- 加密 ZIP 真实密码验证：AES-128/192/256 + ZipCrypto 双模式，错密码不再假成功
- 加密包阅读链路完整打通：封面 + 逐页解密 + inflate
- "忘记密码"功能：详情抽屉一键清除会话密码 ⚠️ **修改加密/解压链路必须读 [经验版 §3](./交接文档-经验思路版.md#3-加密压缩包)**

### 服务端模式

- 运行模式切换：客户端/服务端，入口在 APP能力 页
- 服务信息页：状态摘要、局域网扫描、刷新状态（真实触发重扫）、断开连接
- 远程资源热同步：librarySignature + WebSocket 事件推送，阅读期间延迟刷新
- 服务端挡位热更新：host/port 不变时运行态切换，无需重启
- 远程已下载/资源库/图集：三档展示与切换，回前台/重新进入时强制刷新
- 服务端中文路径修复、favorites handler 优化、_notifyLibraryChanged 收口
- 远程封面路径优先级：cover.jpg > 首张图片，pageSizes 支持图片预占位 ⚠️ **修改服务端路由/扫描/同步必须读 [经验版 §4](./交接文档-经验思路版.md#4-服务端模式)**

### 动态取色与设置交互

- Android 动态取色：只响应亮/暗切换，不再每 6 秒重建 MaterialApp（ANR 修复）
- 设置滑块跟手：三个滑块封装为独立 StatefulWidget
- 外观设置：应用图标 13 色切换、深色模式跟随应用内设置
- Android 启动页重做：全 vector 图标、深色模式适配、双桌面图标修复
- 阅读器状态栏动效：隐藏时淡出动画（改回 immersive 模式） ⚠️ **修改外观/启动/系统栏必须读 [经验版 §5](./交接文档-经验思路版.md#5-动态取色与设置交互)**

### Android 启动与系统 UI

- 阅读器顶部黑条修复：13 个 activity-alias 全补 NormalTheme meta-data
- 启动页无缝过渡：原生启动页 → Flutter bootstrap 壳，"我"页渐进加载
- 系统栏：edge-to-edge、沉浸式显隐、安全区对齐
- 后续新增 activity-alias 必须带 LaunchTheme + NormalTheme ⚠️ **修改 AndroidManifest 必须读 [经验版 §6](./交接文档-经验思路版.md#6-android-启动与系统-ui)**

### 本地回收站与删除链路

- 回收站：真删源 download.db 记录 + 按原 rowid/time 恢复，支持恢复回退
- 跨应用 managed-download 删除/恢复：Shizuku/Root 通道，不误触他人沙箱
- 受限 download.db 不可读判定：区分"不可读"与"真不存在"
- 删除/恢复进度遮罩：防中途退出，执行结束后刷新
- 权限不足直接提示，不再假装成功 ⚠️ **修改删除/回收站/权限链路必须读 [经验版 §7](./交接文档-经验思路版.md#7-本地回收站与删除链路)**

### 已下载页三档隔离

- 本地/聚合/远程·已下载三档加载独立分支，单侧失败不拖垮另一侧
- 页面级超时：本地 8 秒、远程 10 秒，超时进入可重试空态
- Root/Shizuku 列表：优先 download.db 快速构建，封面后台低速补齐
- 远程封面缓存：稳定 provider key + 前 12 项预热 + cacheExtent 一屏
- 详情弹层：标签多时自动高度滚动；移动端 DraggableScrollableSheet ⚠️ **修改已下载页加载/封面链路必须读 [经验版 §8](./交接文档-经验思路版.md#8-已下载页三档隔离)**

### 工具页外显按钮

- "我"页工具卡片一体式设计：底部按钮横向滚动 + 右侧渐变遮罩
- "工具"页支持外显按钮与排序模式：点按显隐 + 长按拖拽排序
- 配置与"我"页共享持久化，默认首批显示：服务信息/本地文件/APP能力/回收站
- 新增工具自动追加到配置末尾，不会漏掉

### 历史同步与图集封面

- 历史页清空/删除后"我"页预览同步刷新
- 远程阅读图片加载进度：真实百分比（需服务端返回 Content-Length）
- 图集封面解码优化：仅限宽不限高 + 像素比余量 + 封面预热
- pageSizes 支持远程图片真实宽高预占位

### 桌面/移动端交互

- 桌面端侧栏顶部被标题栏遮挡修复
- 移动端底部弹层改为 DraggableScrollableSheet（默认 60%，可拖至 80%）
- WindowFrame 主题底色填充

### 其他稳定性修复

- 局域网扫描：秒级快速优先探测
- Android sqlite 运行库缺失修复
- 阅读设置页 keepScreenOn 通道补齐
- Shizuku 授权状态检测修复、Root 会话内复用
- Android 包名统一为 lingxue.picakeep

## 当前仍存在的已知问题

1. Android 前台服务/通知/保活约束适配未彻底关闭
2. 局域网扫描需继续多网卡/多网段真机复核
3. Android 全部文件访问权限需补更多机型验证
4. Shizuku UserService 读大 download.db 偶发 DeadObjectException（当前回退/跳过，不阻塞删除）
5. download.db 残留行对应目录不存在时目录枚举失败日志（当前跳过，不阻塞）
6. 加密包会话密码重启后丢失（需持久化"已解锁+用哪个默认密码"状态）
7. 远程收藏 fetchFavoritesInFolder 串行循环（18 个文件夹 = 18 次请求，可改并行）
8. reader 退出时未主动 abort 未完成图片请求
9. 图片收藏网格列数偶发漂移（推测 settings[74] 存值问题）
